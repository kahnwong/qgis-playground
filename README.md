# QGIS Playground

Ref: <https://courses.spatialthoughts.com/pyqgis-in-a-day.html>

## Pre-requisites

```bash
brew install --cask qgis
```

## Usage

```bash
make build
make start <http://localhost:81> # !!! does not work on m1 mac
```
